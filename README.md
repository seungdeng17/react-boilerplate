# react-boilerplate

### v1.1.6

## Setting

1. root package 설치
```
$ yarn
```

2. boilerplate 초기화
    - 최초 1회 - 포트 설정, 환경 변수 파일 생성 등
```
$ yarn initialize
```

<br />

## Script

- client + server 실행
```
$ yarn dev
```

- build
```
$ yarn prod
```

- proxy 서버 실행 (빌드 파일 테스트용)
```
$ yarn proxy
```
