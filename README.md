# react-boilerplate

### v1.1.7

## Install

- boilerplate 초기화 (의존성 모듈 설치, 포트 설정, 환경변수 생성 등)
```
$ yarn initialize
```

<br />

## Script

- client + server 실행
```
$ yarn dev
```

- build
```
$ yarn prod
```

- proxy 서버 (빌드 파일 테스트용)
```
$ yarn proxy
```

- storybook
```
$ yarn sb
```

<br />
